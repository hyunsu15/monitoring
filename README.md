## 설치방법
backend 터미넬어서 아래 명령어를 수행.
```
docker-compose up -d
```
아래의 사이트로 들어간다
```
http://localhost:8081/
```
주의) 안들어가질경우 mongo-express를 키면 된다.

아래에 있는 명으로 데이터베이스를 만든다.
```
monitoring
```
